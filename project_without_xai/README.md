# Project WITHOUT XAI — Black Box Loan Prediction

This project demonstrates a **black-box** machine learning model that predicts loan approval.
The model gives **NO explanation** for its decisions — just "Approved" or "Denied".

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

## What You'll See

```
Applicant #1: DENIED
Applicant #2: APPROVED
Applicant #3: DENIED
...
```

❌ **No explanation WHY.** This is the problem XAI solves.
